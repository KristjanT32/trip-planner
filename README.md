# Trip Planner
### A new addition to the KrisApps Productivity Suite
Plan your trip's expenses, create an itinerary and set a budget all in one program.
<br>

## An all-in-one trip planning solution brought to you by KrisApps.
### Features
- Expense planning
- Itinerary creation
- Daily and trip budget settings with cost-splitting
- Google Calendar Integration with automatic event generation
- Automatic plan PDF generation with customizable sections
- A handy dashboard with customizable countdowns
